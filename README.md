# CAIC 2100 – Machine Learning Foundations

This repository contains student materials for **CAIC 2100: Machine Learning Foundations**.

The course introduces the foundational concepts, tools, and techniques used in machine learning. Students will work with Python, Jupyter notebooks, pandas, NumPy, Matplotlib, and scikit-learn.

## Course Guiding Principle

**Explore. Discover. Predict.**

Students will explore data, discover meaningful patterns, and use machine learning models to make predictions.

## Software Requirements

Students should install the following software:

* Anaconda Distribution
* Python 3.12
* JupyterLab or Jupyter Notebook

The primary Python libraries used in the course include:

* NumPy
* pandas
* Matplotlib
* Seaborn
* scikit-learn

## Getting Started

1. Install Anaconda Distribution.
2. Download or clone this repository.
3. Open Anaconda Navigator.
4. Launch JupyterLab.
5. Navigate to the appropriate chapter folder.
6. Open `Student_Workbook.ipynb`.
7. Complete the workbook activities in order.
